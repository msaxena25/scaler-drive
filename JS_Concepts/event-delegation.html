* What is Event Delegation?

The idea is that if we have a lot of elements handled in a similar way, then instead of assigning a handler
to each of them â€“ we put a single handler on their common ancestor.
In the handler we get event.target to see where the event actually happened and handle it.

This process is called event delegation.

* Capturing and bubbling allow us to implement one of the most powerful event handling patterns called event delegation.
